# 💸 Who Will Pay Bill

An interactive tool designed to quickly and fairly determine who picks up the check at dinners, parties, or outings. No more awkward debates—let randomness decide!

---

## 🚀 Features

* **Dynamic Input**: Add as many friend names as you like.
* **Instant Selection**: Click a single button to get a random result.
* **Fair Logic**: Uses unbiased randomness to pick the final name.
* **Responsive Design**: Works smoothly on both mobile screens and desktop monitors.


---

## ⚙️ How to Get Started

Follow these steps to run the application locally on your machine.

### Prerequisites
You need a web browser (like Brave, Chrome, or Firefox) installed on your computer.

### Step-by-Step Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```

2. **Navigate to the project folder:**
   ```bash
   cd who-will-pay-bill
   ```

3. **Launch the application:**
   * Open the `index.html` file directly by double-clicking it, or drag it into your preferred web browser window.

---

## 💡 How It Works

1. Enter a name into the input field.
2. Press "Add" or hit the Enter key to add them to the list.
3. Once all names are added, click the **"Pick a Winner"** button.
4. The system randomly selects and highlights the person responsible for the bill!
